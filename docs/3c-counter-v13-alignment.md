# 3C 计数器对齐方案（参考“场次计数器 V1.3”）

## 目标

先把 `3C` 计数器单独做稳，再继续扩到其他路线。

当前阶段只聚焦 4 件事：

1. 自动整轮统计可用  
2. 关键统计口径完整  
3. 误触防护可用  
4. 数据跨会话保留、可手动重置  

## 参考方案映射

### 已落地

- 全自动整轮待命  
  进入大厅或游戏后自动接管会话，回大厅自动结算单把。

- 单把耗时、总把数、总耗时  
  当前会话与历史记录都已保留。

- 总平均耗时  
  现在会基于全部历史场次计算，不只看当天。

- 数据跨会话继承  
  历史记录保存在本地 `data.json`，重开桌宠不会丢。

- 手动重置  
  已补 `重置统计`，会清空计数历史并重新从第 1 把开始。

- 识别间隔设置  
  已支持在桌宠里设置识别间隔，并传给后台监控脚本。

- 锁定防误触  
  已支持锁定计数器窗口。锁定后窗口忽略鼠标，不影响游戏操作。
  当前快捷键：`Alt+Shift+L`
